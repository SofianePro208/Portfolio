My Projects

KALEM Physics

KALEM Physics is an educational web platform designed to make learning physics easier and more interactive for high school and university students. It solves the problem of fragmented, hard-to-understand resources by offering structured lessons, simulations, and quizzes in one place. The platform is built using HTML, CSS, JavaScript, and is currently transitioning to WordPress for easier content management.

QuranWay

QuranWay is a simple and accessible web platform designed to help users stay connected with the Holy Qur’an. It offers three core features in one place: online Qur’an reading, high‑quality audio listening by well‑known Qaris, and a 24/7 live Qur’anic radio stream. The platform is built using HTML, CSS, and JavaScript, ensuring fast performance and a responsive user experience.
