IslamicTab - Extension

Your daily source of tranquility and mindfulness, right in your new tab.

About The Project

IslamicTab is a browser extension designed to replace the default new tab page with a beautiful and functional dashboard. It aims to provide Muslims with a moment of reflection and connection to their faith every time they open a new tab. The interface is clean, minimalist, and focuses on providing essential daily information without being distracting.

Key Features

  • Accurate Prayer Times: Add your location to provide precise prayer times.
  • Hijri & Gregorian Dates: Displays both the Islamic (Hijri) and standard Gregorian dates.
  • Daily Inspiration: Shows a new, inspiring Ayah from the Quran each NewTab.
  • Minimalist Design: A beautiful, clutter-free interface with stunning background images.
  • Lightweight & Fast: Built to be fast and efficient, ensuring it doesn't slow down your browser.

Technology Stack

HTML5 CSS3 JavaScript (ES6+) Web Extension Prayer Times API
